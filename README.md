# Blog Platform

A full-stack blog platform built as part of the final thesis project at ATUSS, Department of VISER.

The application provides a foundation for creating and managing blog content through a modern frontend built with Vue and Tailwind CSS, supported by a PHP backend.

## Technologies

* **Frontend:** Vue.js, Vite, Tailwind CSS
* **Backend:** PHP
* **Styling:** Tailwind CSS

## Project Context

This project was developed as part of the course **Web Application Programming**.

Development and implementation of a web application for a blog platform using modern web technologies.
